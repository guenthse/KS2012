Package praktikum.beispiele

Das Framework.
In der Datei "env.conf" können verschiedene Ablauf-Konfigurationen eingestellt werden.
Eingetragen werden müssen:
- die eigene IP-Adresse
- die eigene MAC-Adresse
- der Gerätename des Netzwerkadapters
Diese Informationen werden vom Kommando "ifconfig -a" geliefert.
- die IP-Adresse der Webservers
Geliefert durch die Kommandos "host hostname" oder "ping hostname".
- die MAC-Adresse des nächsten Ziels im LAN (entweder des Web-Servers selbst oder eines Routers, falls der Web-Server nur über ein oder mehrere solcher "Zwischen-Geräte" erreichbar ist)
Geliefert durch das Kommando "arp -an"

Ein Auszug aus "env.conf": 
...
// Auswahl einer Ablaufumgebung durch setzen der Konstanten "environment" in der Klasse "HttpBeispiel"
// oder Angabe eines Parameters in der Kommandozeile z.B. "-e "unikabel"
environments {
    ...
    // ********* Uni Kabel ********** //
    unikabel {
        ownIPAddress = "141.20.33.23"
        ownMacAddress = "40:6c:8f:11:9e:38"
        // Router 141.20.33.1
        destinationMacAddress = "00:1b:ed:18:1e:00"
        // www.google.de
        httpServerIPAddress = "173.194.69.94"
        // Gerätename (Mac OS X, Monitor-Adapter)
        deviceName = "en4"
    }
    ...
}
